# 列表

- nvm：Node Version Manager
- modules:
  - express: Web框架
  - utility: 一些常用方法
  - body-parser
  - superagent: Http库
  - cheerio： 服务端的jquery
  - eventproxy：异步并发
  - async: 控制异步并发数量
  - should: 断言库
  - mocha: 测试框架
  - istanbul: 测试覆盖率工具（未使用成功）
  - chai: 全栈断言库
  - mocha-phantomjs：在命令行模拟浏览器调用
  - nodemon: 监听代码修改，重启应用，方便调试
  - supertest: 配合集成测试
  - benchmark: 基准测试
  - travis: 持续集成平台
